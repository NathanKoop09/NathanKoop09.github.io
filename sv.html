Slope V

Technical Details

The game currently features three main game play systems: The turn-based battle system, the inventory and skills system, and the cutscenes system.
The battle system has enemy and player objects that have different reactions to death, taking damage, and being notified that it's their turn. This allows for quick scaling to add new enemy types to the game. The status effect system works in much the same way, with a unique id for each status and a flag to indicate a passive effect or an on-turn affect. This system interplayed with the cutscene system allows for a dynamic camera to keep the battle from becoming stale.
The cutscene system consists of a collection of transformations, such as translate to point, rotate continuously, etc. that take a target object to perform the transformation to. These transformations are gathered into shots which then play one after another to form the scene.
The inventory and skills system keeps track of which skills the player has unlocked and communicates this with the battle system. This system stores this information in an optimized structure to reduce the file size of the save file required to store this information.

Project Description

An in development Role Playing Game that explores the interplay of Chaos and Order in one's personal life in a light hearted narrative. The game currently tells the story of Rob, the main player character, and his journey into Crosslink town where he meets the other three members of the party: Beatrice, Natasha, and Cliff.
You can play up to the second boss of the game and explore the world of Crosslink town in the current build of the game.
