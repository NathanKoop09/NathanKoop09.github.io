Eternal Drift

Technical Details

This game was built using a custom made physics engine and collision system to ensure proper behavior at an arbitrarily high speed. The collision algorithm used is the Collide and Slide algorithm using elastic collisions.
To ensure that each race is playable at an arbitrary speed, the game scales the turning rate with the current speed in such a way so that it's always possible to follow the exact same path through a course. The only limiting factor to how fast a player can go is their reaction time.

Project Description

A fast paced racing game with no maximum speed. The more drift boosts you can chain, the faster you'll go, without bounds. Race through four different courses either by yourself or with computer players, or master these courses in time trials mode.
There are four race modes to choose from: Single Race, Tournament, Free Play, and Time Trials. In single race you'll race against four other computer players on a single track.
