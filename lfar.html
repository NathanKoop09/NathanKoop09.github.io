Love Fun and Robots

Technical Details

A game created as part of a weekend long game jam with a theme of "togetherness".
The oil container collectables are saved using Unity's binary formatter which prevents the player from collecting the same container twice.

Project Description

A short and simplistic 2d platformer in which the player needs to control two robots in an attempt to bring them together. Use two different controllers to move the robots through the ten various stages collecting powerups that may help them become reunited once more. Make sure you don't miss the oil containers throughout the levels which you can use to buy cosmetics at the main menu.
